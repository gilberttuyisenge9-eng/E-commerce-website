# 👟 ShoeX Store – E-Commerce Website

A modern responsive shoes shop website built using:

- HTML
- CSS
- JavaScript

This project simulates a simple e-commerce store with cart functionality and local storage.

---

## 🚀 Features

- Responsive design (mobile friendly)
- Dynamic product rendering using JavaScript
- Add to Cart functionality
- Cart counter update
- LocalStorage support (cart remains after refresh)
- Dark Mode toggle
- Clean professional UI

## 🛒 How Cart Works

- Products are stored inside a JavaScript array
- When "Add to Cart" is clicked:
  - Product is added to cart array
  - Cart is saved in localStorage
  - Cart counter updates automatically
